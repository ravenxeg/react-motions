# Changelog

## 1.0.0 (2026-06-26)

Initial public release of `@ravenxeg/react-motions`.

### Features

- **`<Animate>`** — unified component bridging preset animation definitions to `motion/react`
- **`<Magnetic>`** — spring-based hover micro-interaction that pulls elements toward the cursor
- **`fadeIn`** — configurable slide-and-fade entrance (up/down/left/right/none)
- **`scaleIn`** — smooth zoom entrance
- **`blurIn`** — Apple-style blur-and-focus reveal
- **`clipReveal`** — CSS `clipPath` mask reveal (up/down/left/right)
- **`bounceIn`** — elastic spring bounce entrance
- **`flip`** — 3D perspective flip on X or Y axis
- **`swingIn`** — swinging entrance from a top hinge
- **`staggerContainer` / `staggerItem`** — coordinated staggered list entries
- **`animateInView`** — scroll-triggered fade-and-slide default
- Next.js App Router compatible via automatic `"use client"` directives
- ESM + CJS + TypeScript typings via `tsdown`
